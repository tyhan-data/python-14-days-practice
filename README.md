# 14 Days Python Practice

This repository contains Python exercises and mini-projects from **Day 1 to Day 14**.  
Each day focuses on a specific concept to improve your Python skills.

## Structure

- **Day 1:** Variables & Data Types (`day01_variables.py`)  
- **Day 2:** Strings & Methods (`day02_strings.py`)  
- **Day 3:** Lists & Methods (`day03_lists.py`)  
- **Day 4:** Tuples & Sets (`day04_tuples_sets.py`)  
- **Day 5:** If-Else Statements (`day05_if_else.py`)  
- **Day 6:** Loops (for & while) (`day06_loops.py`)  
- **Day 7:** Functions (`day07_functions.py`)  
- **Day 8:** Functions with Return (`day08_return_functions.py`)  
- **Day 9:** String Problems (`day09_string_problems.py`)  
- **Day 10:** List Problems (`day10_list_problems.py`)  
- **Day 11:** Loop + Function Combination (`day11_fibonacci_logic.py`)  
- **Day 12:** Small Projects (`day12_small_projects.py`)  
- **Day 13:** Logic Building (`day13_logic_building.py`)  
- **Day 14:** Mini Project (`day14_mini_project.py`)  

## Features

- Clear and beginner-friendly Python exercises
- Each day as a separate Python file
- Functions, loops, conditionals, and small projects included
- Ready to run in VS Code or any Python IDE
